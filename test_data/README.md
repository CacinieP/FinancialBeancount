# 测试数据说明

此文件夹包含用于测试的示例账单文件，所有数据均为**虚拟示例数据**，不包含任何真实个人信息。

## 文件说明

### sample_alipay.csv
支付宝账单示例文件，包含以下交易类型：
- 餐饮消费（星巴克、麦当劳）
- 交通充值（地铁卡）
- 网购消费（京东）
- 工资收入
- 出行服务（滴滴）
- 会员订阅（腾讯视频）
- 退款处理

### sample_wechat.csv
微信支付账单示例文件，包含以下交易类型：
- 商户消费（星巴克、麦当劳、7-Eleven）
- 转账交易
- 退款交易
- 外卖消费（美团）
- 咖啡消费（瑞幸）
- 话费充值

### sample_bank.csv
银行卡账单示例文件，包含以下交易类型：
- POS消费
- 工资代发
- 退款交易
- 网银支付
- 话费充值

## 使用方法

```bash
# 复制示例文件到 input 文件夹进行测试
cp test_data/*.csv input/

# 运行处理脚本
python example_usage.py
```

## 数据说明

- **日期范围**: 2024-01-01 至 2024-01-15
- **金额范围**: 18.00 元至 8000.00 元
- **账户**: 虚拟账户号码
- **商户**: 使用真实商户名称但交易数据为虚拟
- **姓名**: 使用"张三"等占位符

## 账单格式

这些示例文件展示了各平台的标准账单格式，可用于：
1. 测试解析器功能
2. 验证去重逻辑
3. 演示导出功能
4. 单元测试
